  <name>Task 2: Implement ProcessManager class</name>
  <files>src/ssh/process-manager.ts, src/ssh/process-manager.test.ts</files>
  <behavior>
    - Test 1: startProcess creates process entry with UUID
    - Test 2: startProcess creates temp file paths
    - Test 3: appendOutput appends data to temp file
    - Test 4: appendOutput updates outputSize/errorSize counters
    - Test 5: completeProcess updates status, exitCode, signal, endTime
    - Test 6: getStatus returns process status info
    - Test 7: getStatus returns PROCESS_NOT_FOUND for unknown process
    - Test 8: getOutput reads chunk from temp file with byte offset
    - Test 9: getOutput returns hasMore=true when more data available
    - Test 10: getOutput returns empty data when offset >= file size
    - Test 11: killProcess sends SIGTERM to channel
    - Test 12: killProcess escalates to SIGKILL after 5 seconds
    - Test 13: killProcess with force=true sends SIGKILL immediately
    - Test 14: cleanupProcess deletes temp files and removes from map
  </behavior>
  <action>
    Create `src/ssh/process-manager.ts` implementing ProcessManager class:

    1. **Constructor**: Initialize `Map<string, ProcessInfo>`
    2. **startProcess(host, command, channel, connection)**:
       - Generate UUID with `crypto.randomUUID()`
       - Create temp file paths: `${os.tmpdir()}/ssh-exoman-${processId}.{out,err}`
       - Create ProcessInfo entry with status='running'
       - Return processId
    3. **appendOutput(processId, data, isStderr)**:
       - Append buffer to temp file using `fs.appendFileSync` (sync is OK for small chunks)
       - Update outputSize/errorSize counter
    4. **completeProcess(processId, exitCode, signal)**:
       - Update status: 'completed' if exitCode=0, 'failed' otherwise
       - Set exitCode, signal, endTime
       - Call `connection.end()` to close SSH connection
    5. **getProcess(processId)**: Return ProcessInfo or undefined
    6. **getStatus(processId)**: Return status info (without channel/connection) or PROCESS_NOT_FOUND error
    7. **getOutput(processId, byteOffset, maxBytes)**:
       - Use `Bun.file(path).slice(offset, end).arrayBuffer()` for async chunked reading
       - Return { data: string, totalSize: number, hasMore: boolean }
    8. **killProcess(processId, force?)**:
       - If force=true: send SIGKILL immediately
       - If force=false: send SIGTERM, wait 5s, escalate to SIGKILL if still running
       - Update status to 'killed', set endTime
    9. **cleanupProcess(processId)**:
       - Delete temp files (ignore errors)
       - Remove from map

    User decisions to honor:
    - Hybrid approach: stream output during execution AND persist to temp file
    - Default chunk size: 4KB for pagination (this is in getOutput maxBytes default)
    - SIGTERM, wait 5 seconds, then SIGKILL escalation

    Use the patterns from RESEARCH.md Pattern 4.
  </action>
