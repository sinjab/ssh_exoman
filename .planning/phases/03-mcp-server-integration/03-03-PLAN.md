- **Status:** Failed
- **Reason:** "Cannot parse privateKey: Encrypted OpenSSH private key detected, but no passphrase given"

## Current Implementation

From `src/ssh/client.ts`:
- `connect()` already accepts `passphrase` in `ConnectOptions`
- Passphrase is passed to ssh2 Client when privateKey is loaded

From `src/ssh/executor.ts`:
- Currently reads only `process.env.SSH_PASSPHRASE` (global)
- Passes passphrase to `connect()` function

## Problem

Single global env var `SSH_PASSPHRASE` doesn't support users with different passphrases per host.
